# 「LINE怪盗にゃんこ」を自動で解くプログラム

## 実行例

```
開始:
+---------+
|e G G G  |
|    -    |
|  G G G C|
|         |
|G G G|G  |
|         |
|G G|G G G|
|         |
|G G G e G|
|         |
|G G G|G G|
|         |
|    G G  |
+---------+
怪盗例:
+---------+
|e .←.←.←.|
|  ↓ -   ↑|
|  . .←. .|
|  ↓ ↓ ↑  |
|.←. .|.  |
|↓   ↓ ↑  |
|.→.|. .←.|
|  ↓ ↓   ↑|
|.←. .→C .|
|↓       ↑|
|.→.→.|.→.|
|    ↓ ↑  |
|    .→.  |
+---------+

G = gold
C = cat(1)
. = taken
e = exit

R = ring
c = cat(2)
```
